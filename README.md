# 智能语音药盒系统

基于STM32F103+FreeRTOS的多功能智能药盒，集成多传感器感知、离线语音交互、多模态提醒、数据上传等功能，为用户提供智能化服药管理解决方案。

## 📦 核心功能

- **智能服药提醒**：支持多组闹钟设置，到点触发语音+灯光+舵机开盖多模态提醒
- **取药动作检测**：重量传感器+光敏传感器双重校验，准确识别取药行为，避免漏服误报
- **空盒报警机制**：自动检测药盒剩余药量，余量不足时主动提醒用户补药
- **全离线语音控制**：无需手机APP，纯离线语音即可完成闹钟配置、功能设置等全部操作
- **蓝牙数据上传**：支持服药数据、设备状态上传到手机端，便于健康管理
- **低功耗运行**：传感器周期性唤醒，非关键任务低优先级调度，延长设备续航

## ✨ 创新设计特点

### 1. 融合感知与高可靠容错技术
采用重量传感器+光敏传感器多模态融合算法，双重校验取药动作，大幅降低误报概率；搭配任务看门狗监控、MQTT断线自动重连、蓝牙超时回退等多重容错机制，全链路保障系统运行可靠性。

### 2. 全离线语音交互与无阻塞调度技术
支持完整的离线语音设置流程，无需依赖手机APP即可独立完成闹钟、服药提醒等全部功能配置；基于FreeRTOS抢占式多任务调度机制，实现任务的实时中断与恢复，语音交互全程无阻塞，操作响应流畅无卡顿。

### 3. 分级精准提醒与低功耗优化技术
采用分级提醒策略，闹钟提醒与空盒报警使用差异化语音ID与LED闪烁频率，用户可快速区分告警类型；搭载传感器周期性唤醒机制，非关键任务配置低优先级调度，在保证功能完整性的前提下大幅降低整机功耗。

## 📖 使用说明

### 开机
长按电源键开机，OLED显示欢迎界面，系统自动完成初始化

### 设置闹钟
- **语音方式**：说出"设置闹钟"，根据语音提示依次设置时、分即可
- **按键方式**：短按功能键进入设置模式，上下键调整时间

### 服药提醒
闹钟到达设定时间后，语音播报提醒，LED闪烁，舵机自动打开药盒盖

### 取药确认
取出药物后，系统自动检测重量变化+光敏状态，确认取药完成后关闭提醒

### 空盒提醒
当药盒内药物余量不足时，系统自动播报"请补充药物"提示

## 📅 2025.04 优化更新内容

- **重量精度升级**：从整数升级为浮点型，精度0.1g，适配轻重量药片
- **空盒逻辑优化**：
  - 空盒触发阈值从3g降到0.5g，留0.5g回差防误触发
  - 退出空盒阈值从5g降到2g，轻药片也能正常解除报警
- **滤波优化**：一阶低通滤波，新值占75%权重，兼顾响应速度和平滑度，不会跳字也不会滞后
- **上传逻辑优化**：
  - 上传阈值从1.8g降到0.8g，重量变化响应更快
  - 新增2秒智能同步机制，只有重量和上次上传不一致才上传，不重复浪费流量
  - 双重保险：变化实时上传+定时同步，100%保证屏幕和云端数据一致
- **代码健壮性优化**：统一变量命名，消除语法警告，内存占用优化适配STM32小RAM
