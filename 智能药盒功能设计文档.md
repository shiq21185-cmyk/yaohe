# 智能药盒功能设计文档
## 一、系统概述
智能药盒基于STM32F103C8T6微控制器开发，嵌入FreeRTOS实时操作系统，集成温湿度传感器、压力传感器、光敏传感器、语音模块、蓝牙模块及WiFi模块，实现药品定时提醒、空盒检测、环境监测及数据上传等智能化功能。

## 二、硬件架构设计
### 2.1 核心主控
主控芯片采用STM32F103C8T6，搭载ARM Cortex‑M3 72MHz内核，配备64KB SRAM与512KB Flash存储。系统通过FreeRTOS实现多任务并行调度，各功能模块通过独立任务管理，确保实时性与稳定性。

### 2.2 传感器配置
| 传感器类型       | 型号    | 功能说明                                                                 | 采集周期 |
|------------------|---------|--------------------------------------------------------------------------|----------|
| 温湿度传感器     | DHT11   | 实时监测药品存储环境，温度精度±2℃，湿度精度±5% RH                        | 500ms    |
| 压力传感器模块   | HX711   | 配合称重传感器实现药品重量检测，精度达0.1g，用于空盒检测及取药确认        | 200ms    |
| 光敏传感器       | 通用型  | 部署于药盒开口处，检测环境光照变化，作为判断用户是否取药的辅助依据        | 100ms    |

### 2.3 通信模块
| 模块类型   | 型号    | 通信接口 | 功能说明                                                                 |
|------------|---------|----------|--------------------------------------------------------------------------|
| 蓝牙模块   | HC06    | USART1   | 实现手机APP时间同步及3组闹钟参数配置                                      |
| WiFi模块   | ESP8266 | USART2   | 基于MQTT协议将温湿度、重量及系统状态上传至云平台                          |
| 语音模块   | XRVoice | USART3   | 支持语音播报提醒及语音识别指令（"打开盖子""关闭盖子""设置闹钟"等）          |

### 2.4 执行机构
| 执行机构类型 | 型号 | 驱动方式 | 功能说明                                                                 |
|--------------|------|----------|--------------------------------------------------------------------------|
| 舵机         | SG90 | PWM信号  | 实现药盒盖子的自动开合，开启角度45°，关闭角度135°                        |
| LED指示灯    | -    | PA0引脚  | 用于空盒报警及闹钟提醒时的视觉提示，支持2.5Hz频率闪烁                    |

### 2.5 人机交互
- **OLED显示屏**：128×64分辨率，支持3个显示界面切换：
  1. 界面1：实时显示温度、湿度、重量数值及上传状态
  2. 界面2：大字显示当前时间（蓝牙同步），支持时分秒显示
  3. 界面3：3组闹钟时间设置及音量调节界面
- **按键控制**：4个独立按键（PB6‑PB9），支持界面切换、参数调节及手动停止报警。

## 三、核心功能实现
### 3.1 多任务调度架构（核心优势）
基于FreeRTOS实现8个功能任务的并行管理，**任务间完全无冲突，支持随时中断与恢复**：

| 任务名称       | 优先级 | 功能说明                                                                 |
|----------------|--------|--------------------------------------------------------------------------|
| HX711_Task     | 高     | 重量数据采集与空盒检测逻辑处理                                           |
| DHT11_Task     | 中高   | 温湿度数据采集与上传触发判断                                             |
| Voice_Task     | 最高   | 语音指令识别与响应处理，支持实时中断其他任务执行                         |
| Time_Task      | 中     | 系统时间维护与闹钟触发检测                                               |
| Display_Task   | 中     | OLED界面刷新与显示逻辑处理                                               |
| Key_Task       | 中高   | 按键扫描与按键事件处理                                                   |
| Upload_Task    | 低     | MQTT数据上传与网络通信处理                                               |
| Servo_Task     | 中高   | 舵机控制与盖子开合逻辑处理                                               |

**核心优势：任务间通过全局状态标志与互斥锁实现数据共享，关键数据（时间、闹钟参数）采用互斥锁保护，支持任务抢占与恢复，例如语音说修改闹钟后，可马上询问现在是什么时候，系统自动播放时间后，能自动回到闹钟修改流程继续操作，无需重新发起指令。**

### 3.2 闹钟提醒系统
系统支持3组独立闹钟设置，存储于g_bt_alarms[3]数组结构。闹钟触发条件通过Check_Alarm_Time()函数实时检测，当系统时间（时/分/秒）与任意启用闹钟匹配时触发以下动作：
- 语音播报：当前时间与闹钟匹配时，播放对应提醒语音（如"早上好，记得吃药"）
- 舵机开盖：自动开启药盒盖子，方便用户取药
- LED闪烁：PA0 LED以2.5Hz频率闪烁，提供视觉提醒
- 持续提醒：若光敏传感器未检测到取药动作（光照无变化），每5秒重复播放语音提醒，直至检测到取药或用户手动关闭

**闹钟停止条件：**
1. 光敏传感器检测到光照变化（用户伸手取药）
2. 语音指令"关闭盖子"
3. 按键KEY1‑KEY4任意按下

### 3.3 空盒检测与报警
基于HX711压力传感器实现智能空盒检测，逻辑流程如下：
- 状态判断：当检测到重量从>5g降至<3g（连续2次确认），判定为"空盒"状态
- 自动响应：
  - 自动开启药盒盖子，提示用户补充药品
  - 播放空盒提示语音（语音ID 0x09）
  - LED以2.5Hz频率持续闪烁
  - 每5秒重复语音提醒
- 状态解除：当重新放入药品（重量>5g）或用户手动按键确认后，关闭盖子并停止报警。

**空盒状态通过g_empty_box_alarm_active全局标志管理，与闹钟状态互斥，避免冲突。**

### 3.4 环境监测与数据上传
系统通过独立任务实现环境数据的周期性采集与上传：
- DHT11_Task：每500ms读取温湿度，变化阈值触发上传（温度±0.2℃，湿度±1%）
- HX711_Task：每200ms检测重量，变化阈值2g触发上传
- Upload_Task：通过MQTT协议上传至指定Topic（stm32/temperature、stm32/humidity、stm32/weight、stm32/status），支持状态优先级队列（状态 > 重量 > 温度 > 湿度）
- 数据上传采用防抖机制（500ms去抖间隔），避免网络拥塞。

### 3.5 语音交互系统（非阻塞式设计）
集成XRVoice语音识别模块，支持指令控制，语音任务通过信号量机制与串口中断配合，确保实时响应：
- **支持的语音指令**：打开盖子、关闭盖子、修改闹钟、查询时间、调节音量、停止报警等
- **非阻塞特性**：语音指令处理过程中可随时响应新的指令，例如修改闹钟流程中，用户可随时问"现在是什么时间"，系统播报时间后自动回到闹钟修改状态，继续完成之前的设置
- **多场景响应**：语音指令在闹钟响铃、空盒报警、正常运行等所有状态下均可正常响应，无需等待当前任务完成

## 四、系统工作流程
1. **初始化阶段**：完成传感器校准（HX711去皮）、网络连接（WiFi/MQTT）、蓝牙配对及语音模块初始化
2. **正常运行**：循环执行环境监测、时间维护、显示刷新及按键检测，各任务并行无阻塞
3. **闹钟触发**：检测到设定时间→语音播报→开盖→LED闪烁→等待取药确认→关盖→恢复常态
4. **空盒检测**：重量骤降为0→判定空盒→开盖→语音+LED提醒→等待补药→恢复常态
5. **数据同步**：蓝牙连接时自动校准系统时间，APP可远程修改闹钟参数

## 五、创新设计特点
1. **智能感知融合**：结合重量传感器与光敏传感器双重确认取药动作，避免误报
2. **语音闭环控制**：支持完整的语音设置流程，无需手机APP即可完成闹钟配置
3. **分级提醒策略**：闹钟与空盒报警采用不同语音ID与闪烁频率，便于用户区分
4. **低功耗设计**：传感器采用周期性唤醒机制，非关键任务使用较低优先级
5. **异常安全机制**：任务看门狗监控、MQTT断线重连、蓝牙超时回退等多重容错设计
6. **无冲突多任务调度**：基于FreeRTOS的抢占式调度机制，支持任务随时中断与恢复，语音交互全程无阻塞，操作流畅无卡顿
